# Мои финансы

Локальное веб-приложение для учёта вкладов, накоплений, зарплаты и трат.
Работает полностью в браузере, без сервера и без интернета (кроме
опционального обновления ставок банков). Все данные хранятся у вас на
устройстве — в `localStorage` браузера.

## Запуск

Открыть `index.html` напрямую двойным щелчком браузер, скорее всего,
заблокирует загрузку JS-модулей (ограничение `file://`). Поэтому запускайте
через любой локальный статический сервер, например встроенный в Python:

```
python -m http.server 8000
```

и откройте `http://localhost:8000/` в браузере.

## Перенос данных между устройствами

Специальной синхронизации нет — вместо этого в шапке приложения есть кнопки:

- **Экспорт JSON** — скачивает файл со всеми данными.
- **Импорт JSON** — загружает файл обратно (можно заменить текущие данные
  или объединить с ними).

Перенесите скачанный файл на другое устройство любым удобным способом
(флешка, мессенджер, папка в облаке вроде Google Drive/Яндекс.Диска —
подойдёт что угодно, ведь это обычный `.json` файл) и импортируйте его там.

## Разделы

- **Вклады** — калькулятор вклада (простой и с капитализацией — ежемесячной,
  ежеквартальной, ежегодной), таблица вкладов с текущей и итоговой
  стоимостью, график роста конкретного вклада или суммарный график по всем
  вкладам.
- **Накопления** — цели с целевой суммой, пополнения по датам, прогресс-бар
  и график накопления.
- **Зарплата** — история доходов, график по месяцам.
- **Траты** — история трат по категориям (категории создаются на лету),
  разбивка по категориям, график по месяцам.
- **Общий график** — сводный график по выбранной валюте: вклады, накопления,
  зарплата и траты нарастающим итогом на одной шкале.

Суммы разных валют не смешиваются — везде, где это важно, есть переключатель
валюты.

## Ставки банков (Беларусь)

Раздел «Вклады» показывает актуальные ставки по вкладам белорусских банков и
позволяет одной кнопкой подставить их в калькулятор. Данные берутся не с
сайта каждого банка отдельно (это было бы гораздо более хрупко), а с
агрегатора [myfin.by](https://myfin.by/vklady), который уже сводит
предложения всех ключевых банков в одну таблицу.

**Как вклад попадает в список:** `scripts/parse_rates.py` заходит на три
страницы myfin.by — отдельно для BYN, USD и EUR — и разбирает HTML каждой
карточки вклада (банк, продукт, ставка, срок, доход за срок, капитализация).
Список на сайте разбит на страницы по 20 карточек (`?page=2`, `?page=3`, …);
скрипт читает со страницы служебное поле с общим числом предложений и сам
проходит по всем страницам, пока не соберёт их все — на момент последнего
запуска это 140 предложений в BYN, 35 в USD и 24 в EUR (199 суммарно). Никакого
ручного отбора банков нет — попадает всё, что показывает сам агрегатор для
данной валюты. Если банк или конкретный вклад отсутствует в списке — либо его
нет на myfin.by, либо агрегатор его недавно убрал.

Чтобы обновить ставки:

```
cd scripts
pip install -r requirements.txt
python parse_rates.py
```

Скрипт перезапишет `data/bank-rates.json`. Приложение само подхватит файл при
следующей перезагрузке страницы. Если файла нет — приложение работает как
обычно, просто без блока с готовыми ставками.

Если верстка myfin.by изменится и парсер перестанет находить предложения —
он сообщит об этом в консоль и не тронет старый `bank-rates.json`.
